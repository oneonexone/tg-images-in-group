# Telegram Image Bot

Бот для Telegram, который ищет и отправляет изображения по запросу. 

внимание, у бота абсолютно нет цензуры!

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Получите токен бота:
   - Откройте [@BotFather](https://t.me/BotFather) в Telegram
   - Отправьте `/newbot` и следуйте инструкциям
   - Скопируйте полученный токен

4. Настройте переменную окружения:
   - Windows (PowerShell) - БЕЗ кавычек вокруг токена:
     ```powershell
     $env:BOT_TOKEN="ваш_токен_от_botfather"
     python bot.py
     ```
   - Windows (CMD) - БЕЗ кавычек вокруг токена:
     ```cmd
     set BOT_TOKEN=ваш_токен_от_botfather
     python bot.py
     ```
   - Linux/Mac:
     ```bash
     export BOT_TOKEN="ваш_токен"
     python bot.py
     ```

   ⚠️ ВАЖНО: Не ставьте кавычки вокруг токена при установке переменной!

## Запуск

```bash
python bot.py
```

## Использование

Добавьте бота в группу и отправьте сообщение в формате:
```
%запрос
```

Например:
- `%арбуз` - найдет изображение арбуза
- `%кошка` - найдет изображение кошки
- `%пейзаж` - найдет пейзаж

## Примечание

Бот использует Unsplash Source API, которая не требует API ключа для базового использования. Изображения выбираются случайным образом по ключевому слову.
